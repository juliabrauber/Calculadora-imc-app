# Projeto React Native - Calculadora de IMC

Este projeto foi desenvolvido no início dos meus estudos em React Native. Ele tem como objetivo principal calcular o IMC (Índice de Massa Corporal) a partir dos dados inseridos pelo usuário.

## Sobre o Projeto

- **Propósito:** Praticar conceitos básicos de React Native, componentes, estados e estilização.
- **Funcionalidade:** O usuário informa peso e altura, e o app retorna o valor do IMC e sua classificação.
- **Estrutura:**
  - `src/components/Form`: Formulário para entrada de dados e exibição do resultado.
  - `src/components/Main`: Componente principal da aplicação.
  - `src/components/title`: Exibe o título do app.
  - `assets/`: Imagens e recursos estáticos.

## Como Executar

1. **Pré-requisitos:**
   - Node.js instalado
   - Expo CLI instalado globalmente (`npm install -g expo-cli`)
2. **Instale as dependências:**
   ```bash
   npm install
   ```
3. **Execute o projeto:**
   ```bash
   npx expo start
   ```
4. **Abra no seu dispositivo:**
   - Use o app Expo Go para escanear o QR code exibido no terminal ou navegador.

## Observações

- Este projeto é simples e serve para fins de aprendizado.
- Sinta-se à vontade para modificar e experimentar!
